# Expense Manager (Android)

A simple Android expense management app built with Kotlin and Jetpack Compose.

## Features
- Add an expense with a title and amount.
- Validate user input.
- View a running total.
- See all expenses in a list.

## Run
1. Open the project in Android Studio (Hedgehog or newer).
2. Let Gradle sync.
3. Run on an emulator or Android device.

## Notes
- Data is stored in memory only for simplicity.
- Closing the app clears the current list.
